# H-2A Visa Analysis


This analyzes the number of temporary visas granted to foreign agricultural workers over the last 20 years.


### Tools
	* Anaconda
	* Jupyter Notebook

### Libraries and Packages required
	To analyze the data set, I imported the following libraries.	
	* pandas
	* matplotlib
	* numpy
	* calculate
	* time
	* timeout_decorator
	* geopy
	* geopandas
